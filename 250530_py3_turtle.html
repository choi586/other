🐢 Turtle(터틀)이란?¶

프로그래밍으로 그림을 그리는 방식

“거북이가 화면 위를 기어 다니면서 선을 그린다”

  • forward(100) 앞으로 100만큼 이동
  • left(90) 왼쪽으로 90도 회전
  • right(45) 오른쪽으로 45도 회전
  • penup() 선을 그리지 않고 이동
  • pendown() 다시 선 그리기 시작
  • goto(x, y) 지정한 위치로 이동
  • color("red") 선 색을 빨간색으로 바꿈

# 시어핀스키 삼각형 그리기

def sierpinski(length, depth):
    if depth == 0:
        for _ in range(3):
            forward(length)
            left(120)
    else:
        # 한 변의 중간 거리
        half = length / 2

        # 첫 삼각형
        sierpinski(half, depth - 1)

        # 이동 후, 두번째 삼각형
        forward(half)
        sierpinski(half, depth - 1)

        # 원래 자리로 돌아오고, 다시 이동
        backward(half)
        left(60)
        forward(half)
        right(60)

        # 세번째 삼각형
        sierpinski(half, depth - 1)

        # 원래 자리로 복귀
        left(60)
        backward(half)
        right(60)

# 실행
penup()
goto(400, 100)  # 적당한 시작 위치로 이동
left(150)
pendown()
width(1)
sierpinski(300, 3)  # 길이 300, 깊이 3단계

# 코흐 눈송이 그리기

def koch(length, depth):
    if depth == 0:
        forward(length)
    else:
        length /= 3
        koch(length, depth - 1)
        left(60)
        koch(length, depth - 1)
        right(120)
        koch(length, depth - 1)
        left(60)
        koch(length, depth - 1)

# 눈송이 만들기 (삼각형 형태)
def koch_snowflake(length, depth):
    for _ in range(3):
        koch(length, depth)
        right(120)
